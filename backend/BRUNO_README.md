# Testing Pressure API in Bruno

## 1. Create a New Request
- **Name**: Create Pressure Record
- **Method**: POST
- **URL**: `{{baseUrl}}/device/v1/pressure`

## 2. Headers
Add the following headers:
- `Content-Type`: `application/json`
- `X-Device-Id`: `test_device_bruno`
- `X-Timestamp`: `{{timestamp}}`
- `X-Signature`: `{{signature}}`
- `X-Body-Hash`: `{{bodyHash}}` (required only when `DEVICE_API_REQUIRE_BODY_HASH_SIGNATURE=true`)
- `X-Nonce`: `{{nonce}}` (required only when `DEVICE_API_REQUIRE_NONCE=true`)

> Note: `{{timestamp}}`, `{{signature}}`, `{{bodyHash}}`, and `{{nonce}}` are generated by the script below.

## 3. Body (JSON)
Use this JSON payload (using the valid patient ID found in your DB):
```json
{
  "user_id": "721e584f-f3f9-4beb-81b0-bfc688e487ce",
  "device_id": "test_device_bruno",
  "heart_rate": 80,
  "sys_rate": 120,
  "dia_rate": 70,
  "a": [100, 110, 120, 110, 100],
  "b": [80, 85, 90, 85, 80],
  "measured_at": "2024-02-17T12:00:00Z"
}
```

## 4. Script (Pre-Request)
In Bruno, go to the **Script** tab -> **Pre Request**.
Paste the following JavaScript to automatically calculate the HMAC signature required by the API.

```javascript
const crypto = require('crypto');

// Configuration matches backend/app/core/config.py
  const secret = bru.getEnvVar("deviceSecret");
  const deviceId = bru.getEnvVar("deviceId") || "test_device_bruno"; // Must match payload/header
  if (!secret) {
    throw new Error('Please set Bruno Environment variable "deviceSecret".');
  }

// Generate Timestamp
const timestamp = Math.floor(Date.now() / 1000).toString();
const nonce = crypto.randomBytes(16).toString('hex'); // unique per request

// Pre-request runs before Bruno interpolation, so resolve {{vars}} on raw body text.
const bodyRawFromRequest = req.getBody({ raw: true });
let rawBody = typeof bodyRawFromRequest === "string"
  ? bodyRawFromRequest
  : JSON.stringify(bodyRawFromRequest ?? {});
rawBody = rawBody.replace(/\{\{\s*([^}]+)\s*\}\}/g, (_m, key) => {
  const envValue = bru.getEnvVar(key.trim());
  return envValue !== undefined && envValue !== null ? String(envValue) : "";
});
if (typeof req.setBody === "function") {
  req.setBody(rawBody); // Ensure outgoing payload bytes are exactly what we hash
}

const bodyHash = crypto.createHash('sha256').update(rawBody, "utf8").digest('hex');

// Generate Signature:
// legacy mode: HMAC-SHA256(secret, timestamp + deviceId)
// hardened mode: HMAC-SHA256(secret, timestamp + deviceId + bodyHash + nonce)
const requireBodyHash = true;
const requireNonce = true;
let message = timestamp + deviceId;
if (requireBodyHash) message += bodyHash;
if (requireNonce) message += nonce;
const signature = crypto.createHmac('sha256', secret)
  .update(message)
  .digest('hex');

// Set Variables for Headers
req.setHeader("X-Timestamp", timestamp);
req.setHeader("X-Signature", signature);
req.setHeader("X-Device-Id", deviceId);
if (requireBodyHash) req.setHeader("X-Body-Hash", bodyHash);
if (requireNonce) req.setHeader("X-Nonce", nonce);
```

## 5. Send
Click **Send**. You should verify you receive a `201 Created` response.

## Bruno Environment

Create environment variables (e.g. in `Telemed Device API/environments/local.bru`):

```bru
vars {
  baseUrl: http://localhost:8000
  deviceId: test_device_NEW_006
  deviceSecret: <device specific secret>
}
```

- Keep real `deviceSecret` only in local env files and never commit it.
- Use `local.example.bru` as shareable template.
